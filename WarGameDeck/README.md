# 🗂️ WarGame Deck Portal

Bienvenue dans votre portail de gestion de Decks.

## 🚀 Comment l'utiliser ?

1.  **Ajouter un Deck** : Copiez vos fichiers `.html` exportés depuis "Tech Tree Maker" dans le dossier `Assets`.
2.  **Mettre à jour** : Double-cliquez sur le fichier **`Lancer_WarGame.bat`**.
3.  **Visualiser** : Le script va ouvrir votre navigateur sur `index.html` avec tous vos decks listés et mis à jour.

## 🛠️ Automatisation
Le script `Indexer.ps1` (lancé par le .bat) s'occupe de tout :
- **Conversion** : Il applique automatiquement le thème sombre et moderne aux nouveaux fichiers HTML.
- **Drapaux** : Il détecte le pays selon le nom du fichier pour afficher le bon drapeau.
- **Nouveautés** : Il indexe vos notes de mise à jour.
